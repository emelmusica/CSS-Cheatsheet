CSS Snippet Cheatsheet

This Cheatsheet will allow you to easily find some common syntex for your css applications

FlexBox Row

Use these three properties to create a Flexbox row layout.

            
.container {
display: flex;
flex-direction: row;
justify-content: space-between;}
FlexBox Column

Use this to create a Flexbox column layout.

            
.container {
display: flex;
flex-direction: column;
justify-content: center;}
CSS Grid Layout

Build a 12-column layout using CSS grid.

            
.container {
display: grid;
grid-template-columns: 
repeat(12, 1fr);
gap: <--gap-size-->;
}
Linear Gradients

This will create a background linear gradient from top to bottom.

           
        
.background {
background-image: linear-gradient
(to bottom, #ff0000, #0000ff);
}
Box Transition Glow

Use transition and box shadows to glow on hover.

           
        
glowing-box {
width: 200px;
height: 200px;
background-color: #f1f1f1;
box-shadow: 0 0 10px rgba(0, 0, 0, 0.3);
transition: box-shadow 0.3s ease-in-out;
}
Overlay card with Title

Use position properties and negative margins to raise elements higher than their natural starting point.

            
.element {
position: relative;
top: -20px;
margin-bottom: -20px;
}
